**Context:**
User testing identified critical issues with the Braking Lockup effect: rear wheel lockups were ignored, the manual slip calculation was broken due to a sign error, and the trigger thresholds were hardcoded and too high. Additionally, users could not tune braking vibration intensity independently from road texture harshness because they shared a single "Load Cap" setting.

**Implementation Requirements:**

1.  **Physics Engine (`FFBEngine.h`)**:
    *   **Fix Manual Slip:** In `get_slip_ratio`, use `std::abs(data->mLocalVel.z)` to fix the sign error with forward velocity.
    *   **Rename/Split Load Cap:** Rename the existing `m_max_load_factor` to `m_texture_load_cap` (affects Road/Slide). Add a new variable `m_brake_load_cap` (affects Lockup).
    *   **Add Tuning Variables:** Add `m_lockup_start_pct` (default 5.0), `m_lockup_full_pct` (default 15.0), and `m_lockup_rear_boost` (default 1.5).
    *   **Update Lockup Logic:**
        *   Monitor **all 4 wheels** (not just front).
        *   Implement **Axle Differentiation**: If rear slip > front slip, use 0.3x Frequency and apply `m_lockup_rear_boost` to Amplitude.
        *   Implement **Dynamic Thresholds**: Use the Start/Full % variables to calculate severity with a **Quadratic ($x^2$) ramp**.
        *   Use `m_brake_load_cap` for amplitude scaling instead of the global load factor.

2.  **Configuration (`src/Config.h` & `src/Config.cpp`)**:
    *   Update `Preset` struct to include the new variables (`lockup_start_pct`, `lockup_full_pct`, `lockup_rear_boost`, `brake_load_cap`).
    *   Update `Save`, `Load`, `Apply`, and `UpdateFromEngine` to handle these new keys.
    *   Update `LoadPresets` to initialize defaults (Start=5.0, Full=15.0, RearBoost=1.5, BrakeCap=1.5).

3.  **GUI Layer (`src/GuiLayer.cpp`)**:
    *   **New Group:** Create a new collapsible header **"Braking & Lockup"**.
    *   **Move & Add Controls:**
        *   Move "Lockup Vibration" checkbox and "Lockup Strength" slider to this new group.
        *   Add "Brake Load Cap" slider (1.0 - 3.0x).
        *   Add "Start Slip %" slider (1.0 - 10.0%).
        *   Add "Full Slip %" slider (5.0 - 25.0%).
        *   Add "Rear Boost" slider (1.0 - 3.0x).
    *   **Update "Tactile Textures" Group:**
        *   Rename the existing "Load Cap" slider to **"Texture Load Cap"**.
        *   Update tooltip to clarify it now ONLY affects Road and Slide textures.
