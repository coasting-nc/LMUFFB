# Add a GUI checkbox to disable the addition of Dynamic Load Sensitivity to Tire Grip Estimation #392

**Opened by:** coasting-nc
**Date:** Mar 16, 2026

## Description

Add a GUI option (checkbox) to disable the addition of Dynamic Load Sensitivity to Tire Grip Estimation, so that users with direct drive wheelbases can do tests with and without it enabled, and see if this is introducing any graininess.

This is to help investigate whether adding Dynamic Load Sensitivity to Tire Grip Estimation (introduced with v0.7.181) causes graininess.
