# Kerbs too strong #297

**State:** Open
**Opened by:** coasting-nc on Mar 8, 2026

## Description

There have been multiple user reports that kerb effects are too strong.

This might come from the yaw kick effect. However, reducing the yaw kick effect (there are multiple sliders that can affect it) might also get rid of some useful details.

Investigate if the kerbs signals are also coming from other effects, like Steering Shaft Torque and In-Game FFB.

See if the recent introduction of up-sampling to 400Hz and 1000Hz can be further exploited to reduce the effects of kerbs without losing other details.
